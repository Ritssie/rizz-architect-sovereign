# ⚡ Rizz Architect: Sovereign Edition

The **Rizz Architect** is a premium Social Intelligence Engine built for high-stakes social engineering in modern dating. It combines advanced vision analysis with strategic psychological profiling to turn every interaction into a winning move.

## 💎 Features
- **Triple-A Analysis:** Advanced evaluation of App-Awareness, Atmosphere & Investment, and Anomaly Detection.
- **Sovereign Logic:** Identifies "Shit-Tests" and provides counter-moves to maintain frame control.
- **Archetype Profiling:** Analyzes the target's persona to tailor the tone of voice.
- **Full-Service Logistics:** Provides real-time weather updates, curated venue hotspots, and "Armor" (outfit) recommendations based on location.
- **Closing Radar:** A precision meter that calculates the optimal moment to move the conversation offline.

## 🛠️ Setup & Usage
1. **Deploy:** This app is designed to be deployed on [Streamlit Cloud](https://share.streamlit.io/).
2. **Requirements:** Ensure `requirements.txt` and your logo file are in the root directory.
3. **Activation:** Enter your personal **OpenAI API Key** in the sidebar to initialize the engine.

## 🛡️ Privacy
Your privacy is paramount. API keys are handled session-side and are never stored on the server.
